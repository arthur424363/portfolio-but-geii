## Publier sur GitHub Pages — étapes (5 minutes)

### 1. Créer le dépôt sur GitHub

1. Connecte-toi sur https://github.com (compte `arthur424363`)
2. Clique sur le bouton vert **"New"** en haut à gauche
3. Nom du dépôt : `portfolio-but-geii`
4. Coche **Public**
5. **Ne coche pas** "Add a README file" (on l'a déjà)
6. Clique sur **"Create repository"**

### 2. Uploader les fichiers

Le plus simple : drag and drop dans l'interface web.

1. Sur la page du dépôt fraîchement créé, clique sur **"uploading an existing file"** (lien bleu)
2. Glisse tout le contenu du dossier `portfolio-but-geii` (le `index.html`, le dossier `pages`, le dossier `assets` et le `README.md`)
3. En bas, clique sur **"Commit changes"**

### 3. Activer GitHub Pages

1. Sur la page du dépôt, clique sur l'onglet **"Settings"** (tout à droite)
2. Dans le menu de gauche, clique sur **"Pages"**
3. Sous **"Build and deployment"** :
   - **Source** : *Deploy from a branch*
   - **Branch** : *main* (ou *master*), dossier `/ (root)`
4. Clique sur **"Save"**

### 4. C'est en ligne !

Après 1 à 2 minutes, ton portfolio sera accessible à l'adresse :

```
https://arthur424363.github.io/portfolio-but-geii/
```

C'est l'URL à mettre dans le fichier Excel comme lien vers les preuves.

## Mettre à jour le contenu plus tard

- Modifie les fichiers HTML directement sur GitHub (clique sur un fichier → icône crayon ✏️)
- Ou installe **GitHub Desktop** pour synchroniser le dossier de ton ordinateur avec le dépôt
